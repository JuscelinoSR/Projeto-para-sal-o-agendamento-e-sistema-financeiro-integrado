# Projeto-para-sal-o-agendamento-e-sistema-financeiro-integrado
SaaS de agendamento para salões femininos via WhatsApp. Site elegante + assistente virtual "Ana" com fluxo de 3 cliques, sem digitar. Painel admin com dashboard e QR Code pra conectar WhatsApp. Stack: Next.js, TypeScript, Tailwind, Supabase. Design acolhedor mobile-first.
